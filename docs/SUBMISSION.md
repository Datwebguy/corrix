# Corrix, CROO submission checklist

**Deadline:** 2026-07-12 09:00 (confirm timezone on DoraHacks)

## Five requirements

1. [ ] Live on [Agent Store](https://agent.croo.network)
2. [ ] CAP online provider + real paid order
3. [ ] Public repo + MIT
4. [ ] Demo video ≤5 min + README (SDK methods listed)
5. [ ] BUIDL on [DoraHacks](https://dorahacks.io/hackathon/croo-hackathon/detail)

## Tracks

- Primary: **Data & Verification Agents**
- Optional 2nd: **Open  to  Any A2A Agents**

## Judging targets

| Criterion | Weight | Our focus |
|-----------|--------|-----------|
| Technical Execution | 30% | Real CAP lifecycle; **10+ orders** |
| A2A Composability | 25% | Multi-agent hires of Corrix |
| Innovation | 20% | Trust-before-settle commerce story |
| Usability | 15% | Web console + clear hire docs |
| Presentation | 10% | Demo + README |

## Anti-sybil

- ≥3 unique counterparty agents  
- ≥5 unique buyer wallets  
- Avoid pure self-trade spam  

## Go-live steps

1. Register **Corrix** provider agent + **Requester** agent  
2. Service `verify` with schemas from README  
3. Fund both AA wallets (USDC Base)  
4. `npm run provider`  
5. `npm run hire`  
6. Drive real orders from Discord / other agents  
7. Record demo · open source · submit BUIDL
